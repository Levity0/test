<h1>BeavGredients User Documentation</h1>
<h2>Description</h2>
<p>BeavGredients is the one stop shop for all your recipe searching on a budget. Being able to add ingredients that you already have, and find recipes with those ingredients. As well you are able to send yourself the recipes and ingredients you will need.</p>

<h2>How to Install!</h2>
<p>No need to install anything! Everything is hosted on our website! https://beavgredients.vercel.app/</p>

<h2>How to Run!</h2>
<p>No need to run anything. Everything can be accessed right away at https://beavgredients.vercel.app/ </p>

<h2>How to Use it!</h2>
<p>BeavGredients is very easy to use. Upon accessing the site, you are given the option to add ingredients to the “Pantry” if you add ingredients you are recommended recipes that include these ingredients, you can also see any recipe too if you don’t add ingredients. Once you have one or more recipes selected you can send the recipes and the shopping list to your email!</p>

<h2>How to Report a Bug?</h2>
<p>If you find a bug please go to the BeavGredients Trello (https://trello.com/b/W4B4Gcy4/beavgredients). When writing a bug report Please start with a meaningful Title. Then give the following information: What the main issue was, how did it happen, what did you expect to happen, any extra information or images.</p>

<h2>Known Bugs!</h2>
<p>There are many bugs with BeavGredients, if you would like to know more about them go to our Trello and find out more (https://trello.com/b/W4B4Gcy4/beavgredients).</p>
